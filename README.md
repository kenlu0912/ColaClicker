# Cola Clicker

## Quickstart

```
git clone https://github.com/kenlu0912/ColaClicker.git
cd ColaClicker
java -jar ./target/ColaClicker-1.0.jar
```

## How to build

Dependencies: 
- Maven

```
git clone https://github.com/kenlu0912/ColaClicker.git
cd ColaClicker
mvn compile exec:java -Dexec.mainClass="com.yzu.App" 
```
